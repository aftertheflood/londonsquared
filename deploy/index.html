About London Squared

London Squared is a mapping system created by After the Flood in collaboration with Future Cities Catapult. It was created to make it easier to display data about London boroughs.

The map above uses random Instagram images from each of London’s boroughs. We’ll be posting examples of maps that use the LS system here as we continue to develop it and see it used by others.

Want to make your own maps?

We have developed a simple editor to show London borough data in your web pages. This is in response to all the interest we have had from NGOs and government agencies.
